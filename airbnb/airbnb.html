Airbnb Dataset
Python

This public dataset is part of Airbnb, and the original source can be found on this website. This dataset contains data of guests and hosts of Airbnb in NYC for the year 2019 . I used python libraries like Numpy, Pandas, Matplotlib, and Seaborn to check for any missing values, duplicate values and to visualize the relationship between various parameters in the dataset.
